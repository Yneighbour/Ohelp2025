# 智慧养老系统 - 前端（Vue 3 + Vite，JS）

本目录已从“纯静态页面演示”升级为 **Vue 3 + Vite（纯 JS）** 的前端工程，并开始接入后端 `o-b` 的 REST API。

## 📱 功能页面

### 1. 用户登录页面
- 手机号和密码登录
- 记住密码功能
- 注册链接提示

### 2. 健康数据概览页面
- 显示4项健康指标（血压、心率、血糖、体温）
- 健康状态颜色标识
- 预警消息提醒
- 点击卡片查看详情

### 3. 健康数据详情页面
- 当前数值展示
- Canvas绘制的7天趋势图表
- 历史记录列表
- 健康建议

### 4. 健康预警列表页面
- 预警消息列表
- 预警级别颜色区分（紧急/重要/提醒）
- 未读预警统计
- 点击查看详情

### 5. 健康预警详情页面
- 预警详细说明
- 相关健康数据
- 健康建议
- 联系医生按钮

### 6. 一键紧急呼救页面
- 超大紧急呼救按钮
- 当前位置显示
- 紧急联系人列表
- 快速拨号功能

### 7. 社交动态列表页面
- 热门话题标签
- 动态卡片列表
- 点赞和评论功能
- 浮动发布按钮

### 8. 养老活动列表页面
- 活动分类筛选
- 活动卡片展示
- 报名状态标识
- 参与人数显示

### 9. 活动详情页面
- 活动横幅图片
- 详细信息展示
- 活动介绍和注意事项
- 已报名用户列表
- 底部报名按钮

### 10. 个人中心页面
- 用户信息展示
- 统计数据卡片
- 功能菜单列表
- 退出登录

## 🚀 快速开始

### 1) 启动后端（o-b）
后端默认地址：`http://localhost:8080/api`

### 2) 启动前端（o-f）
在本目录执行：

```bash
npm install
npm run dev
```

默认前端地址：`http://localhost:5173`

### API 地址与代理
- 默认使用 `VITE_API_BASE_URL=/api`（见 `.env.example`）
- 开发期通过 `vite.config.js` 的 proxy 将 `/api` 代理到 `http://localhost:8080`

## 🎨 设计特点

### 适合老年人的设计
- **大字体**：基础字体16px，重要信息18px以上
- **大按钮**：所有可点击元素最小44x44px
- **清晰布局**：简洁明了，避免信息过载
- **颜色对比**：符合WCAG AA标准，对比度≥4.5:1
- **直观图标**：配合文字说明，易于理解

### 响应式设计
- 移动设备优先（< 768px）
- 平板设备适配（768px - 1024px）
- 桌面设备支持（> 1024px）

### 技术特点
- 基于 Vue 3 + Vue Router + Vite 的单页应用
- 样式与旧版静态页面共用 `styles/*.css`，保证视觉一致
- 业务逻辑与路由已迁移为 Vue 组件（见 `src/views` 与 `src/router`）
- 普通列表/卡片页仍然是纯前端渲染，健康详情趋势图继续使用 Canvas 绘制
- 旧版 `scripts/` 下的纯原生 JS 仅保留为设计/对照参考，不再作为入口

## 📝 注意事项

- 后端接口统一返回包裹：`{ code, message, data? }`；前端在 `src/api/http.js` 做了统一解包。
- 登录接口处于“演示模式”：`/auth/login` 失败时也可能返回 `code=200` 且 `data=null`（见后端实现）；前端会按 `data.token` 是否存在判断成功与否。
