## 🚀 使用方法

### 启动后端
- 参考项目根目录 README 中的“后端服务”章节，启动 `o-b`（默认 `http://localhost:8080/api`）。

### 启动前端（推荐）
1. 在 `o-f` 目录执行：

	```bash
	npm install
	npm run dev
	```

2. 浏览器访问 `http://localhost:5173`，即可体验上述 15 个页面（用户端 + 个人中心子页）。

> 说明：`index.html` 与 `scripts/` 下的纯静态实现仅作为设计与功能对照参考，当前演示与联调入口为 Vue 版本（`src/main.js`）。

### 登录信息（演示）
- 如已导入示例数据，可使用数据库中的示例账号（如管理员 `admin/admin`）。
- 若仅做前端 UI 演示，可在后端未启动时观察界面布局与动效，但登录与数据接口不可用。

### 页面导航
- 底部导航栏：健康/社交/紧急/我的
- 点击卡片查看详情
- 返回按钮返回上一页
- 个人中心 5 个菜单可进入子页面

## 📝 技术特点

- 界面与“紫色主题增强版”静态页面保持一致
- 实际运行基于 Vue 3 + Vue Router + Vite 的单页应用
- 页面结构和样式主要由 `styles/*.css` 提供，便于与静态版对照
- 健康详情等趋势图继续使用 Canvas 绘制
- 保留移动端优先的响应式布局与模块化 CSS 设计
